## 1. 迭代版本明细对比

### 阶段一：v1.0 基础 MVP 搭建（2026.7.20）
* **迭代目标**：快速打通系统骨架，实现前后端分离与数据底座。
* **主要交付功能**：
  1. 后端 Spring Boot 3 + SQLite WAL 模式初始化，Flyway `V1.0__init_schema.sql` 自动建表。
  2. 前端 Vue 3 + Vite 5 + Element Plus 脚手架，挂载 DefaultLayout 布局。
  3. 基于 Sa-Token 实现双令牌登录认证与基础 RBAC 菜单权限卡控。
  4. 完成学生主档案（`idx_student`）CRUD 与基础组织机构树（院系/专业/班级）。

### 阶段二：v1.1 业务流程优化与硬卡控闭环（2026.7.21）
* **迭代目标**：深度实现四大业务模块，落实硬性卡控规则与五态状态机。
* **主要交付功能**：
  1. **团员发展 (TY)**：入团申请审批流、推优大会“实到≥2/3、赞成票≥1/2”刚性校验、思想汇报文本查重（>30%打回）。
  2. **社团活动 (ST)**：社团全生命周期管理、招新提前结束（`:finish`）、活动 A/B/C/D 分级审批、现场二维码/GPS 签到。
  3. **社区自治 (SQ)**：宿舍网格管理、巡查日志录入、L1~L4 异常事件分级响应与指导教师结案闭环。
  4. **勤工助学 (QG)**：困难认定前置门禁、岗位发布、考勤打卡与月 40h 工时刚性卡控阻断、系统自动算薪。

### 阶段三：v2.0 AI 赋能与高级创新加分功能（2026.7.22）
* **迭代目标**：实现 MinIO 大文件分片存储与 Spring AI 大模型综测生成，达到答辩加分要求。
* **主要交付功能**：
  1. **MinIO 对象存储**：大文件分片切片上传、预签名 URL 安全预览、Zip 批量打包下载。
  2. **Spring AI / DeepSeek 大模型对接**：提取第二课堂履历，LLM 5秒自动生成学生综合素质测评初稿评语与改进建议。
  3. **人工复核覆写**：辅导员可在前端对 AI 初稿进行覆写保存，最终结果标记“AI初稿+人工覆写”标志。
  4. **ECharts 大屏可视化**：跨模块数据聚合看板、多维度统计与排名。
